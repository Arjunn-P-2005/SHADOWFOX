\# Matplotlib vs Seaborn Visualization Comparison



This project demonstrates a comparison between Matplotlib and Seaborn

using various types of plots:



\- Line Plot

\- Scatter Plot

\- Bar Chart

\- Histogram

\- Box Plot

\- Pie Chart (Matplotlib only)



The goal is to compare:

\- Ease of use

\- Code length

\- Styling

\- Customization flexibility



Run the script using:



python visualization\_comparison.py
