\# Delhi AQI Analysis Project



\## Project Overview

This project performs an in-depth analysis of Air Quality Index (AQI) data for Delhi. 

The analysis focuses on identifying seasonal trends, pollutant correlations, 

anomaly detection, and understanding which pollutants most influence PM2.5 levels.



\## Objectives

\- Analyze PM2.5 trends over time

\- Study seasonal variations in air quality

\- Perform statistical hypothesis testing (ANOVA)

\- Detect anomalies using Z-score method

\- Identify key pollutants influencing PM2.5 using Linear Regression



\## Techniques Used

\- Time Series Analysis

\- Rolling Average Smoothing

\- Seasonal Classification

\- Correlation Analysis

\- ANOVA (Hypothesis Testing)

\- Multiple Linear Regression

\- Data Visualization (Matplotlib \& Seaborn)



\## Key Insights

\- Seasonal differences in PM2.5 levels were statistically tested using ANOVA.

\- Correlation analysis identified strong relationships between pollutants.

\- Regression analysis revealed which pollutants most significantly impact PM2.5 levels.

\- Anomaly detection highlighted extreme pollution events.



\## Technologies Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- SciPy

\- Scikit-learn
